更改配色方法为 Solarized

iTerm2 - Preferences - Profiles - Colors 中选择 Solarized Dark

其他配置

  • 复用上个会话的目录:
    Preferences - Profiles - General - Working Directory - Reuse previous session’s directory

快捷键

  • ⌘ + d: 左右分屏
  • ⌘⇧ + d: 上下分屏
  • ⌘ + Click: 可以打开文件,文件夹和链接
  • ⌘ + enter: 切换全屏

  • ctrl + n/p 选择历史命令

  • alias -s 指定特定的方式打开文件
